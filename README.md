This project is a REST API built with ASP.NET 6 following the MVC architectural pattern (using Controllers and Models, without Views). The goal of this project is to practice the fundamentals of building APIs, handling HTTP requests, and structuring applications using ASP.NET.

The API was developed as part of my learning process while taking the ASP.NET 6 course from Balta.io, where I practiced concepts such as:

Creating controllers and models

Handling HTTP methods (GET, POST, PUT, DELETE)

Receiving data through request bodies

Using dependency injection

Working with Entity Framework

Structuring a clean and maintainable API

This project serves as a hands-on exercise to understand how RESTful APIs work in ASP.NET, focusing on backend development and API request handling.
